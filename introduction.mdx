## Key Features

Operama delivers comprehensive diagnostic solutions tailored for development teams. You integrate it seamlessly into your CI/CD pipelines or monitoring systems. The platform processes metrics, logs, and performance data to generate reports that highlight bottlenecks and optimization opportunities.

<Tabs>
  <Tab title="Basic Usage" icon="play">
    Start with simple queries to test the system.
    ````javascript
    const operama = require('operama-sdk');
    const results = await operama.analyze({ dataset: 'user-logs-2024' });
    console.log(results);
    ````
  </Tab>
  <Tab title="Advanced Queries" icon="code">
    Build complex filters for deeper insights.
    ````javascript
    const query = {
      filters: { status: 'error', timestamp: '>2024-01-01' },
      metrics: ['response_time', 'error_rate']
    };
    const insights = await operama.diagnose(query);
    `````
  </Tab>
</Tabs>

<Steps>
  <Step title="Create Account" icon="user">
    Sign up at the Operama dashboard and verify your email.
  </Step>
  <Step title="Add Data Source" icon="database">
    Navigate to integrations and connect your first data provider, such as GitHub or Datadog.
    ````bash
    operama connect --provider github --token YOUR_TOKEN
    ````
  </Step>
  <Step title="Run First Diagnostic" icon="check-circle">
    Execute a basic analysis command to see immediate results.
  </Step>
</Steps>
